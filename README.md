## Установка и запуск

Клонировать репозиторий:

```bash
git clone https://github.com/Nikiti7/TravelX.git
```

Перейти в папку проета:

```bash
cd travelX
```

Установить зависимости:

```bash
npm install
```

Сборка проекта:

```bash
gulp
```

Деплой проекта:

```bash
gulp docs
```
